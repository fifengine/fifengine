Welcome to the FIFE testing application!
 
Keybindings:
--------------
- prtscr = Make screenshot
- F10      = Toggle console on / off
- ESC    = Quit
 
Commands:
--------------
- clear      = Clear the console window
- help       = This message
- list       = List available tests
- lsrunning  = List the currently running test
- run <test> = Run test
- stop       = Stops the currently running test
